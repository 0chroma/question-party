
This is the Persevere Server.
For more information on Persevere, see 
http://persevere.sitepen.com/.


RUNNING PERSEVERE
=================
From the release, you can run the server with:

   java -jar start.jar

and then point your browser here to see the Persevere documentation and demos:

   http://localhost:8080

You can also view the documentation from the file system by viewing the index.html file.

These directions will start Persevere running on Jetty. You may visit the Jetty website for more information
about configuring Jetty. Persevere can also be run as a webapp in other J2EE servers.
