Persevere JavaScript Client

The Persevere JavaScript Client provides an implementation of the Persistent JavaScript API through HTTP (RFC 2616) with JSON for data interaction with a server.

This library is persevere.js and requires Dojo 1.2. To use the library:
<script src="dojo-path/dojo.js"></script>
<script src="persever-path/persevere.js"></script>

The documentation for the Persevere JavaScript Client can be found at:
http://docs.persvr.org/persevere-javascript-client

This package also includes a Login widget, implemented as a Dojo widget. The Login widget
supports authenticating to Persevere and registering new users.
